## Notes on my personal setup

I'm a [Nix(OS)](https://nixos.org/) enthusiast, you can check out my beloved [Thinkpad X230 configuration](https://github.com/yvan-sraka/X230).
My `~` in handled by [@rycee](https://github.com/rycee) [`home-manager`](https://github.com/nix-community/home-manager) both on [NixOS](https://github.com/yvan-sraka/.config) and [macOS](https://github.com/yvan-sraka/home-manager) (using an [ongoing hack](https://github.com/yvan-sraka/home-brew-manager) to keep things declarative).

Before moving to [Wayland](https://wayland.freedesktop.org) using [Sway](https://swaywm.org/) and [@emersion](https://wayland.emersion.fr/) tools, I was an [xmonad](https://github.com/yvan-sraka/xmonad) user, you can check out this [custom port macOS using X11](https://github.com/yvan-sraka/osxmonad) by [@puffnfresh](https://github.com/puffnfresh) and my [`xmonad.hs`](https://github.com/yvan-sraka/xmonad-config) configuration stolen from [@andreivolt](https://twitter.com/andreivolt).
You can also still found my [old dotfiles](https://github.com/yvan-sraka/dotfiles) from the time I used [Xorg](https://www.x.org/wiki/) with `dwm` and `st` from [suckless](https://suckless.org/).

My code editor in [neovim](https://neovim.io/) and I use ZSH with [Grml configuration](https://grml.org/zsh/).
I use daily [@zx2c4](https://www.zx2c4.com/) [`pass`](https://www.passwordstore.org/) on all my devices, and I moved from Apple keychain data silo using [this script](https://github.com/yvan-sraka/keychain).
